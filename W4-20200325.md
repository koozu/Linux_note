1. 命名空間隔離
2. aufs
3. cgroups -> resources

Docker 指令
* `docker images` : 顯示目前鏡像檔
* `docker ps [option]` : 列出容器
  * `-a` : all 顯示所有容器，包含未運行的
  * `-a -q` : 顯示所有執行的容器ID (CONTAINER ID)
* `docker start [容器ID或名稱]` : 開始、復活容器
* `docker stop [容器ID或名稱]` : 暫停容器
* `docker rm [容器ID或名稱]` : 移除容器
* `docker rmi [鏡像TAG]` : 移除鏡像
