# 附載均衡器 LVS
* 準備三台虛擬機(vm1、vm2、vm3)
* vm1、vm2 -> S1、S2，vm3 -> LVS
## vm1 -> S1：
* 準備三張網路卡(NAT、Host-Only、內部網路)
* 內部網路設定：
  * Address：192.168.1.1
  * Netmask：24
  * Getway：192.168.1.254
## vm2 -> S2：
* 準備一張網路卡(內部網路)
* 內部網路設定：
  * Address：192.168.1.2
  * Netmask：24
  * Getway：192.168.1.254
## vm3 -> LVS：
* 準備一張網路卡(內部網路)
* 內部網路設定：
  * Address：192.168.1.254
  * Netmask：24
