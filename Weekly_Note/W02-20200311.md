## 腳本
>腳本寫時要注意 空白

```shell
#!usr/bin/bash

name=$1

if [ $# -ne 1 ]; then
   echo "`basename $0` domain_name (e.g. `basename $0` www.pchome.com.tw)"
   exit 1
fi

if [ $UID -eq 0 ]; then
   echo "don't use root to run this program"
   exit 2
fi

answer=$(host $name | awk '{print $4}')

echo "$name's ip:" $answer
```
1. 在寫腳本時開頭要加 `#!usr/bin/bash`
